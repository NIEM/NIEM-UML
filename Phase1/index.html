NIEM-UML-3

Alpha-1 Materials

This preliminary specification for NIEM-UML-3 was produced in 2013 based on the NIEM-3 specification and artifacts as then available. It is a basis for future work but many aspects are out of date or incomplete.

  • NIEM-UML-3-Specification-Draft.docx contains the NIEM-UML-2 specification with edits for NIEM-3
  • NIEM-UML-3-OMG-Update201309 contains a presentation on NIEM-3 Changes
  • NIEM-UML-3-Alpha1.4.zip is the draft specification with associated files
  • NIEMUML3-Conops.docx details some of the design considerations
  • Magicdraw directory contains files in MD format
